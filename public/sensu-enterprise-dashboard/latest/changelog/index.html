Enterprise Dashboard 2.16.0 Release Notes

May 30, 2019 — Sensu Enterprise Dashboard version 2.16.0 has been released and is available for immediate download.

CHANGES

Built on Uchiwa 1.7.0:

  • FIX: The API is no longer marked as unhealthy by HTTP status codes less than 500.
